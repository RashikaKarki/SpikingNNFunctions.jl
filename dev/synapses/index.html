Synapse Functions

We provide non-mutating functions for the following synapse models:

  • Dirac-Delta
  • Alpha
  • Excited Postsynaptic Potential (EPSP)
SpikingNNFunctions.Synapse.delta — Function
delta(t, lastspike, q)

Evaluate a Dirac-delta synapse. Use CuVector instead of Vector for GPU support.

Fields

  • t::Real: current time
  • lastspike::Vector{<:Real}: last pre-synaptic spike time
  • q::Vector{<:Real}: amplitude

SpikingNNFunctions.Synapse.alpha — Function
alpha(t, lastspike, q, tau)

Evaluate an alpha synapse. Use CuVector instead of Vector for GPU support.

Fields

  • t::Real: current time
  • lastspike::Vector{<:Real}: last pre-synaptic spike time
  • q::Vector{<:Real}: amplitude
  • tau::Vector{<:Real}: time constant
